موضوع: وراثت تمپلیت‌ها (Template Inheritance) و پیمایش بین صفحات با تگ {% raw %}{% url %}{% endraw %}

۱) اعمال وراثت تمپلیت به صفحات دیگر

پس از ساخت base.html، کدهای مشترک (CSS/JS، هدر، فوتر، ساختار HTML) را از تمپلیت‌های فرزند حذف کنید و فقط محتوای اختصاصی را داخل بلوک‌ها قرار دهید.

نمونه: about.html

{% raw %}{% extends 'base.html' %}{% endraw %}

{% raw %}{% block title %}درباره ما{% endblock title %}{% endraw %}

{% raw %}{% block content %}{% endraw %}
<h2>درباره ما</h2>
<p>این صفحه اطلاعاتی درباره پروژه آموزشی جنگو ارائه می‌دهد.</p>
{% raw %}{% endblock content %}{% endraw %}

نکته: تگ {% raw %}{% extends %}{% endraw %} باید اولین خط در تمپلیت فرزند باشد.

۲) استفاده از تگ {% raw %}{% url %}{% endraw %} برای ناوبری

به‌جای هاردکد کردن مسیرها (مثل <a href="/about/">) از نام مسیرها در urls.py استفاده کنید.

الف) تعریف نام مسیرها در urls.py

from django.urls import path
from . import views

urlpatterns = [
    path('', views.index, name='index'),
    path('about/', views.about, name='about'),
    path('blog/', views.blog, name='blog'),
]

ب) استفاده در base.html

<nav>
  <a href="{% raw %}{% url 'index' %}{% endraw %}">صفحه اصلی</a>
  <a href="{% raw %}{% url 'about' %}{% endraw %}">درباره ما</a>
  <a href="{% raw %}{% url 'blog' %}{% endraw %}">وبلاگ</a>
</nav>

برای صفحهٔ اصلی می‌توانید از <a href="/"> هم استفاده کنید، اما روش توصیه‌شده در پروژه‌های حرفه‌ای استفاده از {% raw %}{% url 'index' %}{% endraw %} است.

۳) مزایای استفاده از {% raw %}{% url %}{% endraw %}

مزیتتوضیح
انعطاف‌پذیریبا تغییر مسیر در urls.py نیازی به اصلاح لینک‌ها در تمپلیت‌ها نیست.
پایداریاز لینک‌های شکسته به علت تغییر مسیر جلوگیری می‌کند.
کد تمیزلینک‌ها معنایی و قابل نگهداری‌تر می‌شوند.

۴) تمرین عملی

  1. نام مسیر برای صفحات index، about و (اختیاری) blog را در urls.py تعریف کنید.
  2. در base.html منوی ناوبری را با {% raw %}{% url %}{% endraw %} پیاده‌سازی کنید.
  3. تمپلیت‌های فرزند را با {% raw %}{% extends 'base.html' %}{% endraw %} و بلوک‌های title و content بسازید.
  4. مسیر /about/ را به /about-us/ تغییر دهید و بررسی کنید که لینک‌ها همچنان صحیح کار می‌کنند.
